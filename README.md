# Portfólio Profissional | Washington Luiz Coutinho

Este repositório apresenta meu Portfólio Profissional Interativo, desenvolvido como parte do Projeto 1 da Fase 1.

O projeto foi construído com HTML5 e CSS3 puro, com foco em identidade profissional, responsividade, organização de código e publicação via GitHub Pages.

## Visão geral

O objetivo deste portfólio é apresentar minha trajetória profissional e acadêmica de forma clara, estratégica e visualmente organizada.

A proposta do site é comunicar minha atuação na interseção entre:

- Ciência de Dados
- Business Intelligence
- Trade Marketing
- Desenvolvimento Web
- Operação, logística e vendas
- Análise de dados aplicada a negócios

O layout foi pensado com uma estética escura, executiva e analítica, utilizando referências visuais de painéis profissionais, dashboards e interfaces voltadas para dados.

## Perfil profissional

Sou Washington Luiz Coutinho, cientista de dados em formação, com graduação em Marketing e experiência profissional em Trade Marketing, BI, implantação, treinamento, operação, logística e vendas.

Minha trajetória combina visão de negócio, pensamento analítico, comunicação, organização e resolução estruturada de problemas.
